# IVS PyCalc

PyCalc je jednoduchá kalkulačka, ktorá podporuje základné funkcie. Bola vytvorená v rámci školského projektu na predmet IVS - Praktické aspekty vývoje software

Prostredie
---------

`Ubuntu 20.04 64bit`

Inštalácia
---------

`wget url`
`sudo dpkg -i pycalc_1.0_amd64.deb`
	
Spustenie
---------

`pycalc`

Odinštalácia
---------

`sudo dpkg -r pycalc`
